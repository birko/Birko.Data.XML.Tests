# Birko.Data.XML.Tests

Unit tests for Birko.Data.XML.

## Test Coverage

- Basic store operations (Create, Read, Update, Delete)
- Async operations
- Bulk operations
- XML serialization/deserialization
- File handling and error cases
- Thread safety

## Running Tests

```bash
dotnet test Birko.Data.XML.Tests
```

## Dependencies

- xUnit
- FluentAssertions
- Moq
- Birko.Data.XML
